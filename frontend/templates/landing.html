Architecture

The 5-Agent LangGraph Pipeline

Every message runs through this orchestrated multi-agent pipeline before you see a response

Node 01
🎯
Intent Classifier

Rule-based detection: explain / solve / example / practice / summarize. Instant, no LLM needed.

→
Node 02
📚
Knowledge Retriever

Subject-specific context + curated resources (Khan Academy, Wolfram, PubChem, LeetCode...).

→
Node 03
🗺️
Curriculum Mapper

Suggests prerequisites and next topics based on grade level — builds a contextual learning path.

→
Node 04
✨
Response Generator

Gemini AI call with full student context: name, grade, learning style, mode and subject.

→
Node 05
✅
Quality Checker

Validates response, extracts follow-up questions, calculates confidence, formats output.

Teaching Modes

5 Ways the AI Can Teach You

Switch modes at any time — each one completely changes the AI's teaching philosophy

📖
Tutor Mode

Full explanation → worked example → Quick Check question. The classic deep dive.

🤔
Socratic Mode

Guiding questions only. Never direct answers — the AI leads you to discover them yourself.

📋
Exam Prep

Key concept → Mnemonics → Common mistakes → Practice question. Built for test day.

🎨
Creative Mode

Stories, metaphors and analogies. Makes complex topics click through narrative.

🔧
Debug Mode

Problem → Root Cause → Step-by-step fix → Key lesson. Perfect for code and math errors.
