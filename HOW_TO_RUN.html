🔧 Troubleshooting
❌ "ModuleNotFoundError: No module named 'flask'"
Virtual environment not activated. Run: source venv/bin/activate (Mac/Linux) or venv\Scripts\activate (Windows), then pip install -r requirements.txt
❌ "Running in demo mode" / no AI responses
API key missing or incorrect. Check your .env file has GEMINI_API_KEY=AIza... (not the placeholder). Restart the server after editing.
❌ "Port 5000 already in use"
Another app is using port 5000. Change PORT=5001 in .env, then open http://localhost:5001
❌ "ModuleNotFoundError: No module named 'langgraph'"
Run: pip install langgraph langchain-google-genai google-generativeai
❌ Database errors / login fails
Delete neurtutor.db and restart. The database recreates itself automatically: del neurtutor.db (Windows) or rm neurtutor.db (Mac/Linux)
❌ Gemini "RESOURCE_EXHAUSTED" error
You hit the free rate limit (15 req/min). Wait 1 minute and try again, or switch to GEMINI_MODEL=gemini-1.5-pro for a different quota bucket.
❌ ".env file not found" or hidden on Mac
Press Cmd+Shift+. in Finder to show hidden files. Or use terminal: nano .env
